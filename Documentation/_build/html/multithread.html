multithread module¶

class multithread.Worker(fn, *args, **kwargs)¶

Bases: PyQt5.QtCore.QRunnable

Worker is used only for small threads which run for a short period like scanning animation etc. Worker thread

Inherits from QRunnable to handler worker thread setup, signals and wrap-up.

Parameters:
  • callback (function) – The function callback to run on this worker thread. Supplied args and kwargs will be passed through to the runner.
  • args – Arguments to pass to the callback function
  • kwargs – Keywords to pass to the callback function
run()¶

Initialise the runner function with passed args, kwargs.

class multithread.WorkerSignals¶

Bases: PyQt5.QtCore.QObject

Defines the signals available from a running worker thread.

Supported signals are:

finished
No data
error
tuple (exctype, value, traceback.format_exc() )
result
object data returned from processing, anything
progress
int indicating % progress
